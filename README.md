# Salesforce ML Service Automation

Real-time ML integration for Salesforce Service Cloud, demonstrating 
Apex triggers, LWC, and Flow automation with external inference endpoints.

## Features
- Apex Trigger calling external ML API on case updates
- LWC displaying churn risk scores and recommended actions
- Flow-based automatic case routing by risk tier
- Named Credentials for secure API authentication
